# 개인성과 관리앱

플랭크린 플래너와 동일한 기능을 가진 개인 플래닝 도구

## 🚀 Features

- 1. 가치 중심 계획 (Foundation)
- -개인 미션 작성: 한 문장으로 표현하는 인생의 목적
- -핵심 가치 3가지: 의사결정 기준이 되는 개인 가치
- -연간 목표: 미션과 연결된 올해의 핵심 목표 설정
- 2. A-B-C 우선순위 시스템
- -A급: 반드시 해야 하는 중요하고 긴급한 일
- -B급: 중요하지만 긴급하지 않은 일 (계획적 실행)
- -C급: 하면 좋지만 미뤄도 되는 일
- -우선순위 자동 제안: 설정된 가치와 목표 기반 추천
- 3. 4단계 계획 시스템
- -월간 계획: 이달의 핵심 목표와 중요 일정
- -주간 계획: 이번 주 Big 3 + 주요 약속
- -일간 계획: 오늘의 A-B-C 할일 + 시간대별 약속
- -시간 블록: 2시간 단위 시간표 (약속 vs 집중업무 구분)
- 4. 투 페이지 일일 관리
- 좌측 페이지 (계획)
- -오늘의 약속 (시간대별)
- -A급 할일 (최대 3개)
- -B급 할일 (최대 5개)
- -C급 할일 (나머지)
- 우측 페이지 (기록)
- -완료 체크
- -간단 메모
- -내일로 이월할 일
- -하루 한 줄 성찰
- 5. 주간 리뷰
- -이번 주 성과: 완료한 A급, B급 업무 확인
- -다음 주 준비: 새로운 우선순위 설정
- 균형 체크: 일과 개인 시간 비율 점검
- -가치 점검: 이번 주 활동이 개인 가치와 일치했는지 확인
- 6. 심플 대시보드
- -오늘 할일 개수: A()/B()/C() 형태로 표시
- -이번 주 진행률: 계획 대비 완료율 퍼센트
- -월간 목표 상태: 목표별 진행 상황 간단 표시
- -다음 중요 일정: 앞으로 3일 내 주요 약속
- 7. 미니멀 입력
- -2분 규칙: 2분 이내 처리 가능한 일은 바로 실행
- -빠른 할일 추가: 한 줄로 간단히 입력 후 자동 분류 제안
- -음성 입력: 이동 중 음성으로 할일 추가
- -템플릿: 반복되는 업무는 템플릿으로 저장
- 8. 포커스 모드
- -A급 업무 집중: A급 할일만 화면에 표시
- -방해금지: 설정 시간 동안 알림 차단
- -25분 타이머: 집중 업무용 포모도로 타이머
- -완료 체크: 간단한 체크로 성취감 제공
- 9. 기본 동기화
- -클라우드 백업: 계획과 기록 자동 저장
- -멀티 기기: 폰, 태블릿, 컴퓨터에서 동일한 정보
- -오프라인: 인터넷 없어도 사용 가능
- -간단 내보내기: 주간/월간 요약을 PDF로 추출
- 10. 습관 관리
- -핵심 습관 3개: 목표와 연결된 일일 습관 설정
- -간단 체크: 했음/안했음 이분법 기록
- -습관 연속 기록: 연속으로 실행한 날짜 카운트
- -주간 습관 리뷰: 이번 주 습관 실행률 확인

## 📦 Installation

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/개인성과-관리앱.git
   ```

2. Navigate to the project directory
   ```bash
   cd 개인성과-관리앱
   ```

3. Install dependencies
   ```bash
   npm install
   ```

4. Start the development server
   ```bash
   npm start
   ```

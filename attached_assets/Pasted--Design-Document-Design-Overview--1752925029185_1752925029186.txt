# Design Document

## Design Overview

### 프로젝트 개요
- **프로젝트명**: 개인성과 관리앱
- **디자인 스타일**: 미니멀
- **디자인 철학**: 단순하고 간결한 디자인

### 디자인 목표
- 사용자 친화적인 인터페이스 제공
- 자기개발에 관심있는 모든 개인가 직관적으로 사용할 수 있는 UI/UX
- webapp에 최적화된 사용자 경험
- 반응형 디자인으로 다양한 기기 지원

## Design System

### 색상 팔레트 (Color Palette)
```
Primary Colors:
- Primary: #000000
- Primary Dark: #ffffff
- Primary Light: #f5f5f5

Secondary Colors:
- Secondary: #e5e5e5
- Background: #9ca3af
- Surface: #6b7280
```

### 타이포그래피 (Typography)
```
Font Family:
- Primary: Helvetica
- Secondary: Arial
- Monospace: System Font

Font Sizes:
- Heading 1: 2.5rem (40px)
- Heading 2: 2rem (32px)
- Heading 3: 1.5rem (24px)
- Body: 1rem (16px)
- Small: 0.875rem (14px)
```

### 간격 시스템 (Spacing)
```
- XS: 0.25rem (4px)
- SM: 0.5rem (8px)
- MD: 1rem (16px)
- LG: 1.5rem (24px)
- XL: 2rem (32px)
- 2XL: 3rem (48px)
```
